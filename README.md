# Power BI Projects

This repository contains Power BI dashboards created for business and HR analytics.

## Projects Included

### 1. Retail Sales Dashboard
- Sales, profit, orders, and margin analysis
- Product category and regional insights
- Delivery and payment mode analysis

### 2. HR Performance Dashboard
- Employee performance and salary analysis
- Attrition tracking and HR KPIs
- Department and role-based insights

## Tools Used
- Power BI
- DAX
- Data Modeling
- Excel / CSV
