# Retail Sales & HR Performance Dashboard (Power BI)

## Project Overview
This Power BI project contains two dashboards built within a single PBIX file to support business and HR analytics.

---

## Dashboard 1: Retail Sales Dashboard

### Business Objectives
- Analyze total sales, profit, and orders
- Identify top-performing products and regions
- Track monthly sales and profit trends
- Monitor delivery status and payment modes

### Key Features
- KPI cards for Sales, Profit, Orders, Avg Margin %
- Product and region analysis
- Monthly trend analysis
- Interactive slicers

---

## Dashboard 2: HR Performance Dashboard

### Business Objectives
- Monitor employee count and average salary
- Analyze attrition and attendance
- Evaluate department-wise performance
- Understand gender and work mode distribution

### Key Features
- KPI cards for employees, salary, attendance, attrition
- Department & role analysis
- Attrition target vs actual
- Interactive filters

---

## Tools Used
- Power BI
